# 书籍翻译器

将您的数字图书转换成任何您想阅读的语言。

## 这个工具能做什么

您是否曾经发现了一本精彩的书,但它只有您不太熟悉的语言版本?这个工具就是为解决这个问题而生的。它可以将您的数字图书(EPUB格式)翻译成您喜欢的语言,让阅读变得轻松愉快,用您最舒适的语言享受阅读体验。

## 为什么要使用这个工具

- **用母语阅读书籍** - 不再为外语书籍苦恼,也不会错过精彩内容
- **原文和译文并存** - 可以同时看到两种语言,非常适合语言学习者
- **保留书籍的原貌** - 翻译后的书籍保持与原书相同的格式、结构和阅读体验
- **支持多种语言** - 可翻译至英语、中文、西班牙语、法语、德语、日语、韩语、葡萄牙语、俄语、意大利语、阿拉伯语和印地语

## 您可以做什么

### 翻译完整书籍

只需从您的设备中选择任何EPUB格式的书籍文件,选择您想要翻译成的语言,然后让工具完成工作。几分钟内,您就会得到一个完整翻译的版本,可以在您喜欢的电子阅读器或阅读应用上阅读。

### 语言学习者的理想选择

由于该工具可以创建双语输出(同时显示原文和译文),它非常适合学生和语言爱好者。您可以阅读目标语言的内容,同时将译文作为参考。

### 自定义翻译方式

有特殊需求?您可以添加自定义指令来指导翻译。例如,您可能希望保持角色名称不变、使用正式或随意的语言,或者以特定方式保留文化典故。

## 使用流程

1. **选择您的书籍** - 从您的电脑中选择任何EPUB文件
2. **选择语言** - 从12种不同的语言中选择
3. **开始翻译** - 观察书籍翻译的进度
4. **阅读和享受** - 在任何EPUB阅读器中打开您新翻译的书籍

## 适合谁使用

- **读者** - 想要享受本语言没有的书籍
- **学生** - 学习新语言,需要有辅助的阅读练习
- **旅行者** - 从不同国家收集书籍并想要理解它们
- **研究人员** - 需要跨越语言障碍获取内容
- **任何人** - 对其他文化的文学作品感到好奇

## 特别之处

与简单的逐字翻译器不同,这个工具能够理解上下文并保持故事或内容的自然流畅。它保留了章节分隔、格式,甚至尽可能在目标语言中保持作者的写作风格。

翻译过程足够智能,可以:
- 在整本书中保持角色名称和术语的一致性
- 尊重原始段落和句子结构
- 保留特殊格式,如强调、引用和列表
- 保持阅读顺序和章节组织

## 开始使用

您只需要:
- 一个想要翻译的EPUB书籍文件
- 几分钟让翻译完成(时间因书籍长度而异)
- 一个电子阅读器或阅读应用来享受您翻译的书籍

开始用您熟悉的语言探索世界各地的文学作品吧!
